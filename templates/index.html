Upload a FLEXCUBE screen

Provide a screen name plus the screen's UIXML and JS files. The app parses them, stores the field mapping, and produces a single meta.yaml you can hand directly to Claude.

View history

What gets extracted

  • From UIXML: every field (name, label, datatype, length, required, LOV), block/grid structure, tabs, buttons.
  • From JS: cross-field dependencies (enables / disables / shows / writes / reads) and inferred validations (regex, length, range).
  • The generated meta.yaml embeds FLEXCUBE conventions (maker-checker, override popup, LOV pattern) so Claude can compose a CLAUDE.md without further prompting.
